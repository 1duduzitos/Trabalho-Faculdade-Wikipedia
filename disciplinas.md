# Disciplinas do 1º Período

## Fundamentos para Engenharia de Software

Disciplina introdutória que apresenta os conceitos básicos da área de Engenharia de Software. Trabalha temas como organização de projetos, trabalho em equipe, uso do Git e GitHub, versionamento de código e noções iniciais sobre desenvolvimento de software.

O objetivo é preparar o aluno para compreender como funciona a construção de sistemas de forma organizada e profissional.

---

## Lógica de Programação (Matemática)

Disciplina focada no desenvolvimento do raciocínio lógico e matemático aplicado à programação. Envolve o estudo de algoritmos, estruturas condicionais, estruturas de repetição e resolução de problemas.

É considerada a base para aprender qualquer linguagem de programação, pois ensina a pensar antes de programar.

---

## Programação 1

Disciplina prática voltada para o aprendizado de uma linguagem de programação. Nela, o aluno começa a desenvolver programas simples, aplicando os conceitos aprendidos em lógica de programação.

São trabalhados conceitos como variáveis, tipos de dados, estruturas condicionais, laços de repetição e funções.

---

## Criatividade e Inovação

Disciplina voltada ao desenvolvimento da capacidade criativa e da criação de soluções inovadoras. Trabalha conceitos como pensamento criativo, resolução de problemas, empreendedorismo e desenvolvimento de ideias.

Tem como objetivo estimular o aluno a pensar fora do padrão e propor soluções diferentes e eficientes.
