# WIKIPEDIA DO CALOURO - Engenharia de Software

## Sobre o Projeto

Este projeto foi criado com o objetivo de desenvolver uma "Wikipedia do Calouro", reunindo informações importantes para alunos do 1º período de Engenharia de Software.

A ideia é ajudar novos estudantes a entender melhor:
Como funciona o curso
Quais são as disciplinas
Dicas para se sair bem no início da faculdade

O projeto foi desenvolvido em dupla como atividade da disciplina de Fundamentos para Engenharia de Software.

---

## Objetivo

Criar um repositório organizado no GitHub simulando uma base de conhecimento para calouros.
